# 06 - Gmail AI Email Assistant

## Goal

Build an AI assistant that processes new Gmail messages and sends a concise summary to Telegram.

## Features

- Detect new Gmail messages
- Extract sender, subject, and body
- Generate an AI summary
- Send the summary to Telegram

## Skills

- Gmail Trigger
- Email Data Processing
- Basic LLM Chain
- Gemini
- Telegram Notification
- Prompt Engineering
