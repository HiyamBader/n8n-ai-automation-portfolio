# 07 - AI Social Media Post Generator

## Goal

Build an AI-powered workflow that generates high-quality social media posts, creates matching AI images, requests human approval, and publishes the approved content to Facebook.

## Features

- Generate social media posts with AI
- Randomly select content type
- Generate AI image prompts
- Generate AI images
- Preview the post in Telegram
- Human approval workflow
- Regenerate content on demand
- Publish approved posts to Facebook

## Workflow

Manual Trigger
↓
Content Configuration
↓
Select Post Type
↓
Generate Social Media Post
↓
Generate Image Prompt
↓
Generate Image
↓
Send Preview to Telegram
↓
Request Approval
↓
Switch

├── 1 → Publish to Facebook
├── 2 → Regenerate Content
├── 3 → Cancel
└── Invalid → Ask Again
