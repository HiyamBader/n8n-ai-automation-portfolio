# 04 - Telegram AI Document Assistant

## Goal

Build a Telegram AI assistant that receives a PDF document and lets the user choose how to process it.

## Features

- Receive PDF
- Validate file type
- Extract text
- Store document session
- Choose action
- Summarize
- Translate
- Explain
